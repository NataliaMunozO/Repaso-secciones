Sección #6
Proceso para crear tablas
Para hacer una tabla utilizamos la etiqueta table
Una tabla se compone de filas , para hacer esto debemos usar la etiqueta tr
También se compone de columnas y para realizar esto debemos usar la etiqueta td
Con la etiqueta table border tenemos la posibilidad de agregarle borde
Estructura de las tablas
Se compone de las etiquetas :
thead
tbody
tfoot
